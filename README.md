# Emergency Satisfaction Survey

مشروع React + TypeScript + Vite + Tailwind لاستبيان رضا مرضى الطوارئ لمستشفى إربد التخصصي.

## التشغيل محلياً

```bash
npm install
npm run dev
```

## البناء

```bash
npm run build
```

## النشر على GitHub Pages

اسم الريبو المستخدم في الإعدادات:

```txt
EmergencySatisfactionSurvey
```

```bash
npm run deploy
```
